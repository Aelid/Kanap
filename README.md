# Kanap

Kanap is an e-commerce website developed as part of the OpenClassrooms Web Developer curriculum.

This project allows users to browse products, view detailed information, manage a shopping cart and place an order through a dynamic interface connected to a REST API.

## 🚀 Live Demo

Frontend: https://luigi-kanap.netlify.app

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Node.js
- Express
- REST API

## ✨ Features

- Browse available products
- View product details
- Add products to the shopping cart
- Modify quantities
- Remove products from the cart
- Submit an order with form validation

## 📦 Installation

### Backend

```bash
cd back
npm install
npm start
```

### Frontend

Open `index.html` in your browser or deploy the `front` folder on a static hosting service.
